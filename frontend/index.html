    Map frame

      The frame controls how the map sits inside the canvas. Templates in the Editor tab are positioned relative to the map, so they sit nicely in whatever margin space the frame creates.

      Freeform map placement (mm)

      Position and size of the map inside the canvas, in millimetres. Map content fits the rectangle with letterboxing if the aspect ratios don't match.

      Margins (%)

      Seeded from the frame preset — tweak to nudge how the map sits in the canvas. (Canvas “Match map” ignores margins.)

      Corners

          Coordinate format

          Used everywhere coordinates appear — the cross labels, the Atlas/Field-notes info strip, and the “+ Add coords” overlay. Decimals apply to the decimal and deg+min formats.

          Lat / Lng grid

          Defaults are neutral grey on light cream — readable on all themes. Pick lighter text + a dark halo for the Blueprint or Ficus themes.

          Cross at a chosen point

          Lat slider moves along the horizontal line (left ↔ right edge). Lng slider moves along the vertical line (top ↔ bottom edge). −1 / +1 push the label past the edge into the margin.

          Marker at the cross

          Cross color is derived automatically from the active theme's leading stroke (road motorway / primary, falling back to coastline).

          Scale bar

          Position is a fraction of the canvas — 0,0 is top-left, 1,1 bottom-right. Auto length picks a 1/2/5 × 10ⁿ value sized to ~20% of the map width.

          Road accent

          Paints every road one color, overriding the theme's road hierarchy — a clean monochrome network in your accent.

          Place importance

          Drops smaller settlements (and intra-city suburbs / quarters) — keeps a large-area map readable. Hide specific labels below.

          Label style

          Applies one color and typeface to all map labels (streets, places, water), overriding the theme. Off = the theme's own label styling.

          Edit individual labels

            Render a map with labels on (Street / Place / Water names) to edit them here — font, size×, color, bold, italic. Drag any label on the canvas to move it (street names ride their road, so dragging nudges them off it); reset position with ⟲.
