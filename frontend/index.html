    When the toggle is off, the canvas uses the theme's .background color (or the frame's preferred color if it has one).

    Map frame

      The frame controls how the map sits inside the canvas. Templates in the Editor tab are positioned relative to the map, so they sit nicely in whatever margin space the frame creates.

      Freeform map placement (mm)

      Position and size of the map inside the canvas, in millimetres. Map content fits the rectangle with letterboxing if the aspect ratios don't match.

        Lat / Lng grid

        Cross at a chosen point

        Lat slider moves along the horizontal line (left ↔ right edge). Lng slider moves along the vertical line (top ↔ bottom edge). −1 / +1 push the label past the edge into the margin.

        Marker at the cross

        Cross color is derived automatically from the active theme's leading stroke (road motorway / primary, falling back to coastline).

        Scale bar

        Position is a fraction of the canvas — 0,0 is top-left, 1,1 bottom-right. Auto length picks a 1/2/5 × 10ⁿ value sized to ~20% of the map width.
